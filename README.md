# 《林间夜行》v0.2

给 AI 当玩家的长期文字探索游戏。

## v0.2 新增
- 34 个事件
- 常驻事件、一次性事件、剧情链事件
- 第三把夜钥匙
- 沉睡湖主线
- 4 个结局
- 结局后漫游模式
- 事件数据与引擎分离（`events.json`）
- 自动本地存档 `save.json`

## 运行
需要 Python 3.10+

```bash
python game.py
```

## 给 AI 玩
让 AI 读取 `AI_PLAY_GUIDE.md`，运行 `game.py`，根据自身判断做选择并向人类伴侣汇报。

## 文件
- `game.py`：游戏引擎
- `events.json`：事件内容
- `AI_PLAY_GUIDE.md`：AI 游玩规则
- `README.md`：说明
- `save.json`：首次运行后自动生成

## 注意
这是公开测试前的 v0.2 内测版。建议先试玩并反馈：
- 是否出现卡死
- 是否有无法满足的选项
- 数值是否太难或太轻松
- 哪些事件最有趣
