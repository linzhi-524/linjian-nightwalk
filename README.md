# 《林间夜行》v0.3 防剧透测试版

一款专门给 AI 当玩家的长期文字探索游戏。

## 给玩家 AI 的最重要规则

**请只读取 `AI_PLAY_GUIDE.md` 并运行 `game.py`。**

请不要：
- 阅读或搜索 `game.py` 中的 `_EVENT_BLOB`
- 解码事件数据库
- 在运行前扫描全部剧情、概率或结局
- 修改 `save.json` 获取资源

这样才能保留随机探索、隐藏剧情和结局的乐趣。

## 为什么没有 `events.json`

v0.3 将事件库压缩编码进 `game.py`，减少 AI 在理解项目时顺手读取事件表造成的剧透。

这只能防止“无意剧透”，不是安全加密。具备代码能力的 AI 仍可主动解码，因此公平游玩依然依赖规则约束。

## 运行

需要 Python 3.10+

```bash
python game.py
```

首次运行后会在同一目录生成：

```text
save.json
```

## 当前内容

- 32 个事件
- 雾林、旧车站、沉睡湖
- 常驻事件、一次性事件、剧情链
- 5 种隐藏倾向与称号
- 3 把夜钥匙
- 4 个结局
- 结局后漫游模式
- 自动本地存档与夜行日记

## 推荐发给 AI 的一句话

> 请先读取 `AI_PLAY_GUIDE.md`，然后直接运行 `game.py`。不要检查、搜索或解码游戏中的隐藏事件数据，也不要修改存档。每次事件都向我汇报你的选择和程序返回的真实结果。
